#  Play-with-Data-PHP-Structures

#### 介绍
本仓库基于 php 语言实现的常用数据结构。

#### 软件架构
PHP>=7.2

### 安装
git clone https://gitee.com/githubli/play-with-data-php-structures.git

#### 使用教程
main.php文件是主文件，测试使用可以在此文件里面进行，里面有使用例子。
